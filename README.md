Hi! Welcome to the MC Trade Sim README.
Thanks for viewing! You can download this as a zip from View Code > Download as zip on the repo page or can use git clone.
Requirements: Python 3.x.
To download for your PC, go to python.org.
Now remove all the spaces from the filename.
Now go to your terminal and type:
cd [game directory]
python3 for Mac/Linux or py for Windows
And then type import MCTradeSim.
Now follow the instructions given in the game. Have fun!
